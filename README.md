# TOKYO Motion - 動画サムネイル画像一括保存ツール

## 使い方

画像を直接ダウンロード（推奨）

```sh
python main.py -q ライブチャット
```

スクリーンショットを保存

```sh
python main.py -q ライブチャット --save-screenshot
```
